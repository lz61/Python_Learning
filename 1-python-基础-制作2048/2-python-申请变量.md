4.python中的列表:

所谓列表,就是能存放多个数据的容器.

我们把列表中的数据称为列表的"元素".

列表的创建:

```
x = [1,2,3] // 列表形式: [数据1,数据2,数据3，...]
```



5.列表的读取:

注意,列表的读取是从下标0开始!

```
print(x[0]) // 1
print(x[1]) // 2
```



6.列表元素的修改:

```
x[0]=1
```



7.列表中的元素不仅可以是数字,也可以是一个新的列表:

```
x=[
	[1,2],
	[3,4]
]
print(x[0])
print(x[1])
// 访问x的第一个元素的第一个元素
print(x[0][0])
```
