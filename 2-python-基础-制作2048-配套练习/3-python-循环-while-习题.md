### 题目 3：列表元素翻倍

### 题目 4：九九乘法表（左下三角）

```
...

```

### 题目 5：斐波那契数列

**要求**：输入一个整数`n`，生成前`n`个斐波那契数列（0, 1, 1, 2, 3, 5...）。
**示例输入**：`n = 5`
**输出结果**：`[0, 1, 1, 2, 3]`



# 分析:

对于第i行:(i: from 1 to n)

首先输出n-i个空格

其次:输出2*i-1个星号
